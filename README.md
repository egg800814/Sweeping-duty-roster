# 掃地排班系統 使用手冊 (V10.0)

歡迎使用本排班系統！本系統旨在透過智慧演算法，公平且快速地分配每日掃地工作。

## 🚀 快速開始

本系統已打包為單一 HTML 檔案，無需安裝任何軟體即可使用。

1.  找到檔案：`掃地排班系統_全功能單檔版.html`
2.  **滑鼠點擊兩下**：直接使用 Chrome、Edge 或 Firefox 瀏覽器開啟即可。
3.  **首次使用**：系統會自動載入公司預設的人員與區域清單。

---

## 📅 每日排班流程

### 1. 勾選出勤人員
在「每日排班」分頁中，勾選今天**有上班**的人員。系統會根據人員的性別、樓層限制自動過濾適合的區域。

### 2. 設定日期與特殊選項
- **排班日期**：預設為今天。
- **隔天是否為假日**：若勾選，系統會自動將「假日加強區域」納入分配（例如：倒空所有垃圾桶）。

### 3. 點擊「開始自動排班」
系統會產出當日的分配結果，您可以直接查看或點擊「列印結果」。

---

## 👥 資料管理教學

### 人員管理
- 您可以新增、編輯或刪除人員。
- **負責人輪值**：系統內建負責人機制，會自動標記本週由誰負責督導（輪值順序可點擊「推進至下週」來切換）。

### 區域管理
- 設定各區域所需的人數、優先度及性別限制。
- **可選區域**：平常不排班，除非您在排班分頁手動勾選（例如：大型會議室）。

### 資料匯入與備份
為了保險起見，建議您定期進行備份：
1.  進入「資料管理」分頁。
2.  點擊「**匯出所有資料**」：會下載一個 `.json` 檔案到您的電腦。
3.  **匯入資料**：若換了電腦，只需將備份檔匯入即可恢復所有名單與歷史紀錄。

## ❓ 常見問題 (FAQ)

**Q：為什麼我換了瀏覽器資料就不見了？**
A：資料是儲存在您目前的瀏覽器中。如果您原本用 Chrome，換到 Edge 開啟會看到初始名單。請使用「匯出/匯入」功能轉移資料。

**Q：清空快取會影響資料嗎？**
A：大部分瀏覽器的「清除快取」不會影響 `localStorage`（本系統儲存資料的地方），但若選擇「清除所有網站資料」則會清空。請務必定期備份 JSON 檔。

**Q：張三今天請假，但系統還是排了他？**
A：請在排班前，取消勾選「出勤人員」清單中張三的名字，再點一次「開始自動排班」。
