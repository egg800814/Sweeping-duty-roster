## 1. 進入管理系統

1. 開啟 `index.html`（排班系統入口）。
2. 點擊右側標有 **ADMIN** 標籤的「🛠️ 今週の朝掃除責任者」卡片。
3. 系統會自動載入最新資料並進入管理介面。

> **💡 提示**：入口頁會自動顯示今日排班負責人姓名。

## 2. 每日排班流程（分頁一）

### 2.1 選擇日期

| 操作 | 說明 |
|------|------|
| **日期選擇器** | 點擊日曆圖示選擇排班日期 |
| **「今天」按鈕** | 快速跳回今天的日期 |

- 切換日期後，若該日已有排班紀錄，系統會自動載入。

### 2.2 勾選出勤人員

人員清單按部門分組顯示，每位人員顯示：
- 姓名 + 性別標示（♂/♀）
- 角色標籤（如「部長」等權重 <= 10 的高階角色，一般人員不顯示）
- 樓層限制標籤（如「限1F」）

**快速操作按鈕：**

| 按鈕 | 功能 |
|------|------|
| **預設** | 恢復為所有「預設出勤=是」的人員 |
| **全選** | 勾選全部人員 |
| **全不選** | 取消所有勾選 |
| **📥 載入雲端出勤** | 從 Google Sheets 拉取今日出勤名單並自動勾選 |

勾選完成後，下方統計列會即時顯示 **今日出勤人數**、**男性人數**、**女性人數**。

### 2.3 設定排班選項

- **🏖️ 隔天放假**：勾選後，指定區域（如④陽台）會多分配 1 人。
- **🔵 可選區域**：預設已勾選，可手動取消不需打掃的可選區域。

### 2.4 執行自動排班

1. 確認出勤人員與選項後，點擊 **🎲 自動排班**。
2. 系統會根據演算法自動分配人員至各區域，考慮：
   - 區域優先級（每日必掃 > 彈性 > 可選）
   - 性別限制（如女廁限女性）
   - 樓層限制
   - 歷史公平性（避免同人連續掃同區域）
3. 排班完成後，結果自動同步至 Google Sheets。

### 2.5 調整排班結果

排班完成後，你可以進行以下調整：

#### 拖曳移動人員
- 直接用滑鼠 **拖曳** 人名標籤到其他區域。
- 系統會自動檢查性別限制（男性不能被拖入女廁）。
- 已確認的人員拖曳時會跳出警告確認。

#### 移除人員
- **點擊** 人名標籤即可從該區域移除。

#### 追加人員
1. 在人員清單中新勾選遲到或臨時加入的人員。
2. 點擊 **🏃 追加人員**，系統會保留現有分配，將新人插入最需要的區域。

### 2.6 儲存與列印

| 按鈕 | 說明 |
|------|------|
| **💾 儲存排班** | 將排班結果存入本地紀錄（瀏覽器 localStorage） |
| **🖨️ 列印** | 開啟列印預覽，可列印當日排班表 |

> **⚠️ 重要**：每次調整後記得點擊「💾 儲存排班」，否則重新整理後會遺失變更。

### 2.7 確認狀態即時監控

- 排班結果中的人名標籤會即時反映打掃人員的確認狀態。
- 系統每 **5 秒** 自動從 Google Sheets 輪詢最新確認狀態。
- 已確認的人名會標記為 ✓ 並變色。

## 3. 人員管理（分頁二）

點擊頂部的 **👥 人員管理** 分頁進入。

### 3.1 檢視人員清單

表格欄位：

| 欄位 | 說明 |
|------|------|
| 部門 | (動態選項)如：管理部 / 營業部 / 新事業部 / 技術部 |
| 姓名 | 人員姓名 + 限制標籤 |
| 性別 | ♂ 或 ♀ |
| 角色 | (動態選項)如：一般 或 部長 |
| 狀態 | 在職 或 停用 |
| 是否輪值 | 標記為「不輪值」的主管將不會被加進打掃負責人清單 |
| 預設出勤 | 是否預設每日出勤 |

### 3.2 新增人員

1. 點擊 **＋ 新增人員**。
2. 填寫以下欄位：
   - **部門**（下拉選單）
   - **姓名**
   - **性別**（男/女）
   - **角色**（下拉選單）
   - **參與負責人輪值**（是/否）
   - **預設每日出勤**（是/否）
   - **樓層限制**（無限制/僅1樓/僅2樓）
   - **排除區域**（多選，按住 Ctrl 可複選）
3. 點擊「新增」完成。

### 3.3 編輯人員

點擊該人員列右側的 ✏️ 按鈕，修改資料後點擊「儲存」。

### 3.4 刪除人員

點擊該人員列右側的 🗑️ 按鈕，確認後刪除。

> **💡 提示**：若有特定主管不參與排班或打掃，可在角色給予高權重（數字小於 10），並於人員設定將「參與負責人輪值」設為否。

## 4. 區域管理（分頁三）

點擊頂部的 **🏢 區域管理** 分頁進入。

### 4.1 檢視區域清單

表格欄位：

| 欄位 | 說明 |
|------|------|
| 區域名稱 | 清掃區域名稱 |
| 樓層 | 全 / 1F / 2F |
| 優先級 | 🔴 每日必掃 / 🟡 彈性 / 🔵 可選 |
| 性別限定 | 不限 / ♀ 限女 / ♂ 優先男 |
| 人數 | 最少~最多人數，假日前是否+1 |

### 4.2 新增區域

1. 點擊 **＋ 新增區域**。
2. 填寫：區域名稱、樓層、優先級、性別限定、最少/最多人數、假日前是否加人。
3. 點擊「新增」完成。

### 4.3 編輯 / 刪除區域

- ✏️ 編輯：修改區域設定後儲存。
- 🗑️ 刪除：確認後刪除該區域。

### 4.4 優先級說明

| 優先級 | 行為 |
|--------|------|
| 🔴 **每日必掃** | 每天一定會被排入，人數不足時也會優先分配 |
| 🟡 **彈性** | 出勤人數足夠時才排入 |
| 🔵 **可選** | 需在排班頁面手動勾選才會納入排班 |

## 5. 輪值設定（分頁四）

點擊頂部的 **🔄 輪值設定** 分頁進入。

### 5.1 查看輪值順序

列表顯示所有負責人的輪值順序，標有「本週」的為當前負責人。
每位人員旁顯示其代理人（= 輪值表中的下一位）。

### 5.2 操作說明

| 操作 | 方法 |
|------|------|
| **切換本週負責人** | 點擊該人名列即可立即切換 |
| **調整順序** | 點擊 🔼 / 🔽 按鈕上下移動 |
| **推進下一週** | 點擊 **⏩ 推進下一週** 按鈕 |
| **設定基準日** | 點擊 📅 按鈕，將該人設為今日起算的自動輪值基準點 |

### 5.3 自動輪值機制

設定基準日後，系統會根據當前日期與基準日的週數差自動計算本週負責人，無需每週手動推進。

> **⚠️ 注意**：設定基準日後，請記得到「資料管理」分頁匯出 JSON，確保設定被儲存。

## 6. 常見問題

### Q：排班後忘記儲存怎麼辦？
排班結果已自動同步至 Google Sheets，但本地紀錄（歷史紀錄）需點擊「💾 儲存排班」才會保存。

### Q：某人臨時請假怎麼辦？
1. 在出勤人員清單中取消該人的勾選。
2. 重新點擊「🎲 自動排班」即可。

### Q：排班結果中某區域顯示「⏭️ 今日跳過」？
代表出勤人數不足以覆蓋該區域。可以透過追加人員或手動拖曳解決。

### Q：如何查看過去的排班紀錄？
切換日期選擇器到過去日期，若該日有儲存過排班，系統會自動載入。
